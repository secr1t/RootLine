# RootLine (rl)

A minimal system monitoring and control utility written in Rust.

Designed as a lightweight alternative to traditional task managers and tools like `htop`, providing a unified CLI interface for inspecting system state and running diagnostics.

It exposes a simple command-based interface for accessing CPU, memory, network, and process information in a structured and fast way, optimized for Linux environments and DevOps workflows.

Focused on performance, minimal overhead, and direct access to system-level metrics without external dependencies.
